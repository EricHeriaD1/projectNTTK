Код страницы на первом этапе
На данном скриншоте показана структура страницы на первом этапе. Присутвует класс main - весь размер страницы в браузерном окне, window1 - верхняя шапка старницы, класс top_adapt отвечает за картинку в классе window1. Далее идет класс window2 - втарая составляющая, заполняющая оставшуюся часть страницы. Блок wondow2 включает 3 класса - left, middle, right, которые отечают за левую, центральнуя и правую часть страницы соответсвенно. В класс middle помещенны блоки box и pic, которые в будущем будут заполнены картинкой этапа и подписью к ней.
На данном скриншоте представлен css код для первого этапа. Именно тут прописаны все свойства блоков описанных выше.
